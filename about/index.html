Aboat. Aboot. About.

This website was created to document my thesis writing shenanigans using LaTeX. Whilst at it, I thought might as well learn about css and html—somewhat akin to hitting three birds in one stone, and methinks this project managed to combine all three.

About this website

  • Made with love, lots of head scratching, trials, and errors
  • Ongoing project (aka work in progress)
  • Hosted on GitHub through GitHub Pages
  • Validation
  • The logo was made from combining Source Serif Pro (Light), and Charter (Bold Italic) in Photoshop 7. It was inspired a wee bit by this.

Typefaces

  • FF Bauer Grotesk – Titles
  • FF Nuvo – Headings, menu links, and footer text
  • Source Serif Pro – Body text
  • Source Code Pro – Code
